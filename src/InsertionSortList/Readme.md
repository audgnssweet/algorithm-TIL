## 삽입 정렬

https://leetcode.com/problems/insertion-sort-list

---

<h5>내 풀이법</h5>

    1. 삽입정렬을 구현했는데, boolean 플래그도 썼고
    로직이 너무 복잡해보이는 경향이 있다.

---

<h5> 정석 풀이 </h5>

    1. 변수를 많이 사용했다.
    -> root (루트)
    -> pre (이동하는 녀석)
    -> cur (이번에 정렬할 녀석)
    -> next (다음에 정렬할 녀석)

    2. 이렇게 하니, 로직이 가벼울 수밖에 없다.

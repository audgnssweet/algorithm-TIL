## 여러 정렬 리스트의 병합

https://leetcode.com/problems/merge-k-sorted-lists

---

<h5>내 풀이법</h5>

    1. 연결리스트의 특성과, 정렬되었다는 점을 고려했다.
    2. 두 연결리스트의 합을 k개 반복한다고 풀이했다.

<h5>정리</h5>

    1. 속도가 엄청나게 느린편이었다.
    2. 리스트 병합에도 더 빠른 방법이 있었다.

---

<h5>정석풀이</h5>

    1. 우선순위 큐를 이용 (자료구조)
    2. 우선순위 큐는 중복을 허용했다.
    2. 속도는 평균이었다.
